# 🏪 SuperStore Sales Dashboard & Forecasting

## Project Overview
Created an advanced Power BI dashboard with 15-day sales 
forecasting for a SuperStore chain using historical sales data 
and Power BI's built-in analytics features.

## Problem Statement
The store management needed both a performance overview and 
a short-term sales forecast to plan inventory and staffing. 
This dashboard solves both needs in one view.

## Tools Used
- Power BI Desktop
- Microsoft Excel (data cleaning)
- DAX (time intelligence functions)
- Power BI Forecasting (built-in analytics pane)

## Key Features
- 15-day sales forecast using time series analysis
- Sales and profit by category, sub-category and region
- Payment mode and shipping mode analysis
- Year-over-year (YOY) sales comparison
- KPI cards: Total Sales, Total Profit, Total Orders, Avg Delivery Days

## Key Insights
- Standard class shipping used in 58% of all orders
- Office Supplies category had highest order volume
- West region consistently outperformed others in profit
- Forecast shows 12% sales growth expected in next 15 days
